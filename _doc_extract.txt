All Scenarios Build Guide — Scenarios 2 to 22
1. How to Use This Document
This document covers Scenarios 2 through 22. Scenario 1 (Baseline) is complete. Each scenario section contains: exact ETL clone source, all tile changes with exact values, pre-run checklist, post-run SQL verification, and UI wizard settings.
Credit Usage Rules:
Use tile PREVIEW for all debugging — 0 credits
Use SQL tiles on existing datasets to check numbers — 0 credits
Only run full ETL when ALL previews confirmed — 1 credit per run
Expected total runs for Scenarios 2-11: 10 runs (1 per scenario)
Expected total runs for Canada Scenarios 12-22: 11 runs (1 per scenario)
NEVER run ETL to test a change — always preview the specific tile first
